更新日志

2020-01-05 · 615字 · 3 min read

2024-06-09

  • 修改了部分样式，去除了多个页面中的 css class: animated fadeIn，避免页面刷新时抖动
  • 修改了文章 TOC 目录的宽度样式, 只限制最小/最大宽度，使其能够自适应内容宽度
  • 统一 post 文章和 diary/gallary 页面的回到顶部/底部按钮样式和位置
  • 简化了 明亮/黑暗模式切换代码，删除了 ldbutton.js，使用 Vue 来控制样式切换。但刷新会导致页面闪屏，因为 Vue 执行完 mounted 后才会正确计算当前模式；解决：使用 js 函数来提前完成模式的初始化，避免闪屏

2023-12-20

  • 添加时光/说说功能，数据来源于 Memos
    • 由于评论部署在 vercel，加载慢，说说评论需要在原始 Memos 页面打开评论 (更新：添加了页面评论，可以直接在当前页面访问)
    • 说说服务器部署在 Azure，可能需要代理才能正常访问
  • 将仓库同步到 github，利用 github actions 进行博客同步
    • 之前使用 Mac 监控脚本同步博客，gridea 每次预览可能会触发导致站点 url 是 localhost，并且有时任务触发不及时
  • 添加 gridea-search 插件，对应添加了搜索框，调整了一下菜单界面

2023-12-12

  • 更新评论系统 twikoo 版本到 1.6.27
  • 更新样式，删除友链菜单，直接在关于页面添加，保留了部分 css 样式
  • 优化，使用 v-cloak 避免出现 post 页面的文章目录闪烁问题

2022-07-06

  • 修改样式，添加暗亮模式切换
  • 修改默认样式查询方式，简单样式适配
  • 修改代码主题为 vs-dark

2022-03-20

  • 添加文章列表中的标签的显示，设置标签样式(默认没有 border-indigo-500 等样式)
  • 优化布局

2022-02-27

  • 添加代码样式，增加行号和复制选项

2022-02-25

  • 修改主题配置，添加友链菜单，优化了页面效果
  • 添加导航菜单
  • 页面样式移动和修改

2022-02-20

  • 更新页面样式，添加版权和说明（改变字体可能会导致中文目录不能正确跳转）
  • 更新归档页面，添加日期信息和每篇文章的相关描述

2022-02-10

  • 简单使用 Rocky 主题
  • 修改主页信息的描述以及 footer 信息
  • 使用 Twikoo 评论
